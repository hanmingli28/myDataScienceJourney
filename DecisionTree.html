Discussion

The key feature of the dataset is "AcresBurned". Decision tree #1 is created solely based on "AcresBurned", and it produces the most accurate result with a prediction accuracy of 1, which suggests that the label of the dataset is probably created based on the feature "AcresBurned".

Decision tree #2 and #3 were constructed to investigate how parameter cp would affect the accuracy of prediction. Decision tree #2 has cp = 0.05, while decision tree #3 has cp = 0.03. A greater cp would result in a less complex tree structure, which tend to have less overfitting in theory. However, in this case, a smaller cp (tree #3) with a more complex tree structure is actually more accurate.

Discussion

The common root node is "wildfire" for all three trees. The goal is to investigate how parameter max_depth would affect the accuracy of prediction. To control the variable, all three decision trees are based on entropy. Tree #1 has max_depth = none, which means there is no limit on the depth of the tree. Thereofre, the tree will extend until all leaf nodes are pure. Tree #2 and #3 has max_depth = 5 and 10, respectively. Based on confusion matrices, tree #1 has the highest accuracy of 0.75. Tree #3 has a slight worse accuracy of 0.69, while tree #2 has the lowest accuracy of 0.44 among the three. The result suggests that if max_depth is too low, it would cause the decision tree to be inaccurate. However, it should be noted that if max_depth is too high, it would make the tree structure very complex, which could lead to overfitting.
