Discussion

Naive Bayes model output interpretation: First of all, the A-prori probabilities indicates that the training dataset is not balanced, with 75% labeled FALSE and only 25% labeled TRUE. The conditional probabilities suggest that attributes AcresBurned, Injuries, PersonnelInvolved and StructuresDestroyed all contribute significantly to label prediction since the mean value of data with label FALSE is significantly different from that with label TRUE for these attributes.

The confusion matrix shows that the model behaves well in predicting FALSE class data, but not as good when predicting TRUE class data since the model only correctly predicted half of TRUE class data. The visualization also showcase the same result. The inaccuracy when comes to predicting TRUE class data is probably due to the fact the the training data is highly imbalanced, with only 1/4 being TRUE class data.

Conclusion

In conclusion, to determine whether a wildfire is a major indicent or not, we should look at the area that the fire has spreaded accross, the number of injuries associated with the fire, the number of personnel involved in the process as well as the number of structures being destoryed. A wildfire is very likely to be categorized by the authority to be a major indicent when it has burn through about 12k acres of land, has more than 1 injury, has more than 156 people involved, and lastly, has more than 86 structures destroyed because of the fire.

Discussion

Naive Bayes model output interpretation: First of all, the purpose of the NB model is to predict based on text, whether a piece of new article is on the topic of wildfire or not. If yes, then that piece of news is class WF. If no, then it falls into class NOT_WF. The dataset is imbalance with more NOT_WF then WF data. The output of the NB model shows that it has a 0.8 percision in predicting NOT_WF class data, while only a 0.62 percision when comes to WF class data. This is largely due to the imbalanced training dataset.

Conclusion

In conclusion, to achieve a better accuracy in determining whether a piece of news article is about wildfire or not, one should sample balanced data.
